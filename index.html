PhishNet

Text-Based Fraud Transaction Detection

What is PhishNet?

PhishNet is a text-based backend system designed specifically to flag fraudulent transactions. It does not use a user interface or handle phishing links — instead, it focuses purely on detecting irregular or suspicious transaction activity in financial data.

Our Mission

We aim to provide a lightweight, backend-first fraud detection pipeline for transaction data. PhishNet identifies fraudulent activity patterns and flags potential threats, allowing institutions to take action quickly. Alerts are triggered through backend logic and sent via SMS.

About This Project

PhishNet is a Capstone project built in collaboration with Capital One as part of our academic program at UW-Madison. The system is fully text-driven and built for scalability, transparency, and security — with a sharp focus on fraud detection in transactional datasets.
